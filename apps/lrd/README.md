# apps/lrd

Liquid Rocket Design (LRD) application.

## Features

- Dimensions liquid rocket engines based on input design criteria (chamber pressure, thrust, O/F ratio).
- Performs thermochemical equilibrium calculations for both frozen and shifting expansion.
- Outputs detailed motor characteristics including throat/exit diameters and chamber volumes.

## Usage

```bash
lrd
```

Reads design parameters from `input.lr`.
