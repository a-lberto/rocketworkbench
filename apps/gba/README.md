# apps/gba

Grain Burn Analysis (GBA) tool.

## Features

- Simulates the 2D cross-sectional burning of a propellant grain.
- Supports different grain shapes (nuclear, rectangular, cross, round).
- Generates Matlab/Octave scripts for visualizing the grain regression over time.
- Computes burning perimeter and surface area at each step.

## Usage

The application is currently configured via source code in `gba.c`. It outputs `.m` files for visualization.
