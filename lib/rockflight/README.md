# librockflight

Flight dynamics and ballistics library.

## Features

- **Aero**: Aerodynamic coefficient calculations.
- **Atmos**: Atmospheric models (density, pressure, temperature vs altitude).
- **EOM**: Equations of Motion for rocket trajectory simulation.
- **Mass**: Vehicle mass properties and staging.

## Constants

- Defined in `rockflight.h` (Gravity, Earth mass/radius, angular velocity).
