# apps/prop

Interactive propellant property explorer.

## Features

- CLI prompt (`PROP>`) for querying thermodynamic data of individual species.
- Supports commands for:
  - `enthalpy`, `entropy`, `gibbs`, `specific_heat`.
  - `molar_mass`, `info`.
  - `list` (search species by formula).

## Usage

```bash
prop
```

Type `help` at the prompt for a list of available commands.
