# apps/rockflight

Rocket flight trajectory simulator.

## Features

- 6-DOF (Degrees of Freedom) trajectory simulation.
- Atmospheric modeling and aerodynamic coefficient integration.
- Supports multi-stage rockets.
- Outputs data in a format suitable for Octave/Matlab plotting.

## Usage

```bash
rockflight -f mission.conf
```

Requires a mission configuration file specifying rocket properties and initial launch conditions.
