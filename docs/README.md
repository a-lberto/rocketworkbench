# RocketWorkbench 

## Applications

Suite of tools for rocket propulsion analysis, design, and flight simulation.

### Propulsion & Thermochemistry

- [**cpropep**](./cpropep): Main CLI for chemical equilibrium and rocket performance (NASA RP-1311).
- [**prop**](./prop): Interactive shell for exploring thermodynamic properties of chemical species.
- [**analyser**](./analyser): Post-test analysis of thrust-time data from static fires.

### Motor Design

- [**lrd**](./lrd): Liquid-fuel Rocket engine Design and dimensioning.
- [**mcp**](./mcp): Solid Motor Combustion Performance (internal ballistics simulation).
- [**gba**](./gba): Grain Burn Analysis (2D cross-section regression).
- [**grainanalysis**](./grainanalysis): Specialized tools for star grain geometry generation.

### Vehicle Design & Simulation

- [**profiler**](./profiler): Mass, CG, and inertia profiling of rocket assemblies.
- [**rockflight**](./rockflight): Flight trajectory simulation (ballistics and 6-DOF).

### Web Interfaces

- [**cpropep-web**](./cpropep-web): CGI web interface for Cpropep.
- [**lrd-web**](./lrd-web): CGI web interface for LRD.
